Instructors who are currently teaching through Blackboard in Spring 2017:

Canvas will be the new Learning Management System (LMS) for Clemson University starting on May 17, 2017. All future courses will be taught through Canvas.

There are several critical actions that will be required of instructors before Blackboard becomes unavailable on June 30, 2017:

  1. Course Migration

    We have bulk migrated Blackboard courses into Canvas for the following semesters: Fall 2015, Spring 2016, Summer 2016. Course name and ID remains the same in Canvas as it was in Blackboard. If you need other courses to be migrated to Canvas, please submit a form here with the course name and course ID.

  2. Export a Blacboard Course

    Use the Export Course tool to create a package file containing all of the course content for reuse in Canvas or to serve as your own backup. This tool does not include any users or user interactions with the course. For instructions on how to export your course, visit this webpage.

  3. Download Grades from the Grade Center

    Grades must be downloaded by instructors. All instructors should download copies of the Blackboard course grades, especially if you have students with incomplete grades. Downloaded grades can be used in a spreadsheet program such as Microsoft Excel. For instructions on how to do this, visit this webpage.

    Please make sure to adhere to the Registrar's FERPA guidelines for storing sensitive data.

    NOTE: Entering grades into the Blackboard Grade Center is NOT the official process for reporting student grades. Official final grades are entered in iRoar (iRoar > Faculty Services > Final Grades).

  4. Students with Incomplete Grades or Assignments

    Student submissions and grades will not be transferred from a Blackboard course to Canvas. We recommend that you download assignments, tests or other data for your own records and set up a Canvas course for those students to complete their course work if it will not be completed before May 17, 2017.

  5. Support

    If you have any questions or concerns about the transition process, please send an email to ITHELP@clemson.edu.
