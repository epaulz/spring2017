What Is the Respondus LockDown Browser?

The Lockdown Browser is a secure browser for taking tests in Blackboard. It prevents students from printing, copying, going to another URL, or accessing other applications during a test. If a Blackboard test requires that the LockDown Browser be used, students cannot take the test with a standard web browser. The LockDown Browser should only be used for taking Blackboard tests. It should not be used in other areas of Blackboard.

Is There a Cost Associated with the Respondus LockDown Browser?

Clemson licenses the Respondus LockDown browser for use in all areas of activity–teaching, research, and public service.  There is no charge to Clemson users.

How Do I Download and Install the Respondus LockDown Browser?

  1. Navigate to the Respondus LockDown Browser Installation website.
  2. Download the installer and follow the onscreen instructions to complete the install.
